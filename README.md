# Node.js

##Ejecucion

*Ejecutando Node.js en (CLI) command-line-interface*

    $ node
    
    >console.log("Hello, world!");
    Hello world
    >undefined
    
*Podemos ejecutar node.js desde un archivo en nuestro directorio de nuestro equipo*

        console.log(Hello, Wolrd!); //file.js
        
        $ node file.js
        Hello, Wolrd!
